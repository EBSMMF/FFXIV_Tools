## 注意事项

可以根据自己的cp自行修改[制作处理-开始制作]中的触发器条件，将658修改为657/656/655，

若进行上述修改，可能极小概率无法使用出比尔格的祝福

自动补充料理的方式,默认为未开启,[发生异常-buff]

使用方法:将触发器开启.将魔匠放置在8号.料理放置在9号.
